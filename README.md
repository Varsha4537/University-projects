## 🔧 How to Run

For Python projects:

```bash
cd project-folder
pip install -r requirements.txt
python main.py
````

For C projects:

```bash
cd project-folder
gcc program.c -o program
./program
```

For React project:

```bash
cd first-react-project
npm install
npm start
```
